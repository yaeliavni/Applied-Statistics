# HMC

This project demonstrates Random Walk vs Hamiltonian Monte Carlo
using an interactive website and supporting visualizations.

## Live Demo
 [Click here](https://yaeliavni.github.io/HMC/HMC.html) to open the interactive lab.

## Mode Selection
Basic Comparison: 专 转 砖 转  注转 砖  拽专 -HMC.
Advanced Analysis: 住驻拽 转 住住 驻专 转专,   转拽 砖 爪注.
Treasure Hunt Game-  专转: 爪 转专转 砖 专转 专砖 砖注 -10 "爪专转" (专 注 住转专转 ) 爪. 
HMC Mind Reader: 爪 转 转 砖 砖 HMC  转,  砖 专, 爪注转 住 转 拽.
